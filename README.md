# 📚 Book Management System

## 📖 Project Overview

This project is a **Book Management Web Application**.

The system allows users to manage a collection of books by performing basic **CRUD operations** (Create, Read, Update, Delete).

The application is built using **Angular for the frontend** and **ASP.NET Web API with C# for the backend**. The backend provides **RESTful APIs** that are consumed by the Angular frontend.

---

# 🚀 Features

* Add a new book
* View all books
* Update existing book details
* Delete books
* RESTful API integration
* Interactive API testing using Swagger

---

# 📦 NuGet Packages Used

The backend project uses the following **NuGet packages**:

* **AutoMapper** – Object-to-object mapping
* **Microsoft.EntityFrameworkCore** – ORM framework for database operations
* **Microsoft.EntityFrameworkCore.Design** – EF Core design-time tools
* **Microsoft.EntityFrameworkCore.SqlServer** – SQL Server database provider
* **Swashbuckle.AspNetCore (Swagger)** – API documentation and testing

---

# 📊 Book Model

The Book entity contains the following properties:

* **Id** – Unique identifier
* **Title** – Book title
* **Author** – Author name
* **ISBN** – Book ISBN number
* **PublicationDate** – Date the book was published

# ⚙️ How to Run the Project

## 1️⃣ Clone the Repository

```
git clone https://github.com/ChameeraKavinda/BookManagement-.net-project-with-Angular.git
```

---

## 2️⃣ Run the Backend (ASP.NET)

1. Open the backend project in **Visual Studio**
2. Restore NuGet packages
3. Run the project

The API will run on:

```
[https://localhost:7209/api/Book]
```

Swagger API documentation will be available at:

```
https://localhost:7209/swagger
```

---

## 3️⃣ Run the Frontend (Angular)

Open the frontend folder in **VS Code** and run:

```
npm install
ng serve
```

Then open the browser:

```
http://localhost:4200
```
